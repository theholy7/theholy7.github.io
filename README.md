## theholy7.github.io

Personal website for Jose Antunes, built with Jekyll and hosted on GitHub Pages. Uses the [Hydeout](https://github.com/fongandrew/hydeout) remote theme.

### Running locally

Requires Ruby. Install dependencies and start the local server:

```sh
bundle install
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`.

### Publishing

Committing to the `master` branch automatically triggers a GitHub Pages build and deploys the site.
